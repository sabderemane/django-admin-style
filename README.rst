
====
django-admin-style
=====
|pypi| |build| |coverage|

Create a django admin more sexy than the default one

Installation
--------------
Need to have Python and django installed before.
You can install this package with pip: `pip install djangocms-admin-style`


Quick start
-----------

1. Add "django_admin_style" to your INSTALLED_APPS setting like this::

    INSTALLED_APPS = [
        ...
        'django_admin_style',
    ]

2. Run ``python manage.py migrate`` to update style.

3. Start the development server and visit http://127.0.0.1:8000/admin/
   to view the administration.
